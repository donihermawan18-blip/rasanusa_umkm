# rasanusa_umkm
Rasa Nusa adalah toko UMKM yang menghadirkan camilan tradisional dan kuliner Nusantara autentik. Dibuat dari bahan pilihan berkualitas dengan resep warisan turun-temurun, kami menyajikan cita rasa khas Indonesia yang otentik, higienis, dan cocok dijadikan oleh-oleh maupun camilan keluarga.
